There are 4 main ways to make changes to how things work

1. Mod Config 
   - Used in general configuration of mod operation
   - Found in game in the Mods Menu
   - Needs an overhaul
2. Recipes
    - Can be added manually or through KubeJS integration
    - For current recipes see [here](./src/generated/resources/data/nuclearcraftneohaul/recipe)
3. DataMaps
    - Used to alter what blocks/fluids some machines can use
    - List of options be be found [here](./DataMaps.md)
4. KubeJs
    - Used to add recipes as well as new blocks to certain machines
    - More info [here](./KubeJS.md)
