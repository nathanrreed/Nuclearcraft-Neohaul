# Processors

These machines process materials, usually by the use of energy, to convert materials from one type to another.

| Processor                           | Description                                                                                                                                          |
|-------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| <ItemLink id ="nuclear_furnace" />  | Quickly smelts items, using uranium ingots or dust as fuel                                                                                           |
| <ItemLink id ="alloy_furnace" />    | Combines base metals together into alloys                                                                                                            |
| <ItemLink id ="assembler" />        | Combines components into a complex product                                                                                                           |
| <ItemLink id ="centrifuge" />       | Separates isotopes out of fluid materials                                                                                                            |
| <ItemLink id ="chemical_reactor" /> | Houses reactions between fluids, allowing for chemical reactions to occur                                                                            |
| <ItemLink id ="crystallizer" />     | Precipitates solids from a fluid                                                                                                                     |
| <ItemLink id ="decay_hastener" />   | Forces radioactive materials to decay, resulting in the formation of a different isotope or material. Radiation is released during the decay process |
| <ItemLink id ="electrolyzer" />     | Splits fluids into their constituent parts                                                                                                           |
| <ItemLink id ="fluid_enricher" />   | Adds materials to fluids                                                                                                                             |
| <ItemLink id ="fluid_extractor" />  | Extracts fluids from materials                                                                                                                       |
| <ItemLink id ="fluid_infuser" />    | Adds fluids to materials                                                                                                                             |
| <ItemLink id ="fluid_mixer" />      | Mixes fluids together                                                                                                                                |
| <ItemLink id ="fuel_reprocessor" /> | Eextracts materials from depleted fuel, allowing for the isolation of useful material from depleted fuel                                             |
| <ItemLink id ="ingot_former" />     | Forms molten materials into ingots or gems                                                                                                           |
| <ItemLink id ="manufactory" />      | A machine that has many uses, mainly in processing materials                                                                                         |
| <ItemLink id ="melter" />           | Melts down materials into a liquid form                                                                                                              |
| <ItemLink id ="pressurizer" />      | Processes materials under immense pressure                                                                                                           |
| <ItemLink id ="rock_crusher" />     | Crushes rocks to produce mineral dusts                                                                                                               |
| <ItemLink id ="separator" />        | Separates materials into their constituent parts                                                                                                     |
| <ItemLink id ="supercooler" />      | Lowers the temperature of fluids                                                                                                                     |
